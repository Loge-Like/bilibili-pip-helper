·Bilibili视频浮窗定位助手：页面画中画悬浮播放，更沉浸的体验；页面智能定位，告别浏览器放大后的手动拖拽滚动条。优化B站观影体验。
  
  
✨ 功能特点
  
🎬 画中画系统

    视频悬浮播放：将视频窗口脱离页面，随意拖动位置，看视频、刷评论两不误

    滚动自动缩小：向下滚动页面超过阈值，视频自动缩小至角落，不挡视线；向上滚动超过阈值，自动恢复居中

    点击外部缩小：点击视频外区域自动缩小（可开关），点击缩小后的窗口恢复居中

    ESC键智能控制：画中画状态下，按一次ESC缩小到角落，再按一次关闭画中画

    尺寸微调：通过画中画按钮旁的+/-按钮临时调整窗口大小

    背景遮罩：可调节浓度的毛玻璃效果（支持性能模式关闭动画）
      
   
🎯 页面定位系统

    水平居中：当浏览器窗口放大至需要横向滚动时，自动将视频内容水平居中，再也不用手动拖拽滚动条

    多种触发模式：可选择仅在页面加载时居中、仅点击宽屏时居中、或两者兼备

    垂直偏移：点击宽屏/网页全屏时，可自动滚动到指定垂直位置（如避开顶部弹幕）

    退出全屏居中：退出全屏后自动恢复居中位置

    自动宽屏：页面加载时自动点击宽屏按钮
   
   
⌨️ 其他实用功能

    防止空格键下滑：按空格键不会滚动页面，方便暂停/播放视频

    自动网页全屏：页面加载时自动进入网页全屏（与自动宽屏等互斥）

🛠️ 使用说明

    脚本安装后，在B站视频页面会自动在底部控制栏添加一个画中画按钮（📺图标）。点击即可开启/关闭画中画模式。

    所有配置项均可在Tampermonkey菜单中调节（脚本管理器图标 → 弹出菜单 → 选择对应项）。
